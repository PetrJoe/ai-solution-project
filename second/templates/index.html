Project Overview

This project demonstrates the implementation of a University Course Management System using both PostgreSQL (object-relational) and MongoDB (NoSQL document store) databases.

Key Features

  • Document-oriented features in MongoDB (embedded documents, arrays)
  • Complex queries in both database systems
  • Comparison of data modeling approaches
